# Recipo - Recipe Origins Explorer

A web application that helps users discover the historical origins and cultural significance of various dishes from around the world.

## Features

- **Recipe Exploration**: Browse recipes from different cuisines
- **Cultural Context**: Learn about the historical origins of dishes
- **Interactive Forms**:
  - Submit new recipes
  - Write reviews and ratings
- **Local Storage**: Persistent data storage for submitted content
- **Responsive Design**: Mobile-friendly interface
- **Search Functionality**: Find recipes by name or cuisine type

## Technology Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage API
- Google Material Icons
- Google Fonts (Poppins, Merriweather)
